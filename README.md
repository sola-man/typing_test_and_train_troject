# Typing Speed Test

A beautiful, real-time typing speed test with WPM calculation and accuracy tracking.

## Features
- Real-time WPM and accuracy calculation
- Color-coded character highlighting
- Multiple test paragraphs
- Responsive design

## Live Demo
[Click here to test your typing speed](https://sola-man.github.io/typing-speed-test)

## Local Usage
Simply open `index.html` in any modern browser.

## Technologies
- HTML5
- CSS3 (Gradients, Animations)
- Vanilla JavaScript
